⚙ System Data Rules & Access Control

🏷
Branch ID Tagging All records include Branch ID at creation. No data exists without a branch reference.
🔒
Role-Based Access Each session is bound to a role. Menus, queries, and mutations are filtered by role.
🌐
Super Admin — Global Can read and write across all branches. No Branch ID filter applied.
🏥
Branch Users — Isolated Branch Admin, Manager, Therapist: data scoped to their Branch ID only.
📋
Audit Logging All create/update/delete actions are logged with User ID + Branch ID + Timestamp.
🔐
Session Security JWT tokens carry role + branch claims. API validates on every request.
